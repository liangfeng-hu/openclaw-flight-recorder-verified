# Security Policy

## Scope
This repository is a research preview / PoC for local observability and log integrity.
It is NOT a malware scanner and does NOT provide security guarantees.

## Reporting a Vulnerability
If you believe you found a security issue:
1) Open a GitHub issue with minimal details, OR
2) If you prefer private reporting, contact the maintainer via GitHub profile.

Please do not include secrets (tokens/keys) in reports.

## Threat Model Notes
- No installer scripts (curl|sh)
- No editor extensions shipped
- Prefer local-only usage and sharing only digests/summaries
